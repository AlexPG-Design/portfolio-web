Política de Privacidad

Este sitio web respeta y protege la privacidad de los usuarios. No se recopilan datos personales sin el consentimiento explícito del visitante.

En caso de que se habiliten formularios de contacto, sus datos serán utilizados únicamente para responder a las consultas recibidas. No se compartirán con terceros ni se utilizarán con fines comerciales.

Este sitio puede utilizar cookies técnicas para mejorar la experiencia de navegación. No se emplean cookies de seguimiento ni de terceros con fines publicitarios.

El usuario puede configurar su navegador para rechazar las cookies, aunque esto podría afectar a algunas funcionalidades del sitio.

Este sitio está alojado en GitHub Pages, lo que implica que ciertos datos técnicos (como la dirección IP o el tipo de navegador) pueden ser registrados por el servidor de alojamiento con fines estadísticos y de seguridad.
